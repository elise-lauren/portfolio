Slumber Sleepover Kit

A packaging concept for a kids’ sleepover subscription kit.

Identity Design, Package Design, Product Design

Ready for the best slumber party ever? Slumber is a concept for a colorful and fun set of packaging for a children's monthly sleepover subscription kit service.

Client

Student Project

Duration

4 Weeks

Software

Adobe Illustrator, Photoshop, Indesign

The Slumber Sleepover Kit contains everything you need for a fun, exciting sleepover. Each box contains a collection of themed snacks, as well as a movie of the month and a themed set of kids pajamas.
